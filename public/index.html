Perfil

Hola!!! Me encanta ayudarte a crear aplicaciones web profesionales, usando los últimos estándares y tecnologías para que no te quedes atrás.

Si te estás preguntando ¿qué es un Frontend?, es el encargado de maquetar la estructura semántica del contenido (HTML), codificar el diseño en hojas de estilo (CSS) y agregar la interacción con el usuario (JavaScript). En pocas palabras, es la persona que hace la parte del software(página web o web App) que interactúa con el o los usuarios.

Crear una Web App para que se vea y se sienta tal y como la imaginaste es mi misión.
